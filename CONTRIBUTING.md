## Contributing

Thanks for your interest in contributing!

### Getting Started

1. Fork the repository and create a feature branch.
2. Follow the setup instructions in `SETUP.md`.
3. Keep changes focused and small when possible.

### Development Notes

- Backend: `poetry run uvicorn api.main:app --reload --port 8000`
- Frontend: `cd frontend` then `npm run dev`

### Pull Requests

- Describe the motivation and scope of the change.
- Include screenshots or logs if UI or runtime behavior changes.
- Ensure new features include basic tests or manual verification steps.
